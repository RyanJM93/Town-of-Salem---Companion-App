What's new:

  • AUT's written with JavaFX as a user interface toolkit can now be tested.
  • You can now view tasks from HP ALM in the ITE and automatically add test results to them after a test execution.
  • You can now report to ALM systems (Jira, Bugzilla, HP ALM) after running a test via the testexec.
  • You can now track changes to Test Cases and Test Suites in your project.
  • You can see property information about components in the Object Mapping Editor when you collect a component in the Object Mapping Mode.

The list above is just an overview. For a more complete guide to new features, read the release notes.
